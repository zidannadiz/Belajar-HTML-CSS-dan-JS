ID (Identifier) adalah atribut yang digunakan untuk memberikan identitas unik pada elemen HTML. Setiap elemen dapat memiliki satu ID, dan ID harus unik dalam sebuah dokumen HTML. ID sering digunakan untuk mengidentifikasi elemen tertentu dalam CSS atau JavaScript.

ID (Identifier) adalah atribut yang digunakan untuk memberikan identitas unik pada elemen HTML. Setiap elemen dapat memiliki satu ID, dan ID harus unik dalam sebuah dokumen HTML. ID sering digunakan untuk mengidentifikasi elemen tertentu dalam CSS atau JavaScript. 
<div id="header">
  <h1>Selamat Datang di Situs Saya</h1>
</div>

Dalam contoh di atas, elemen <div> memiliki ID "header". Kita dapat menggunakan CSS untuk memberikan gaya khusus pada elemen ini, atau menggunakan JavaScript untuk memanipulasi elemen tersebut berdasarkan ID-nya.
